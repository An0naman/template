Security Guidelines
Recommended Safe Types:
  • Images: jpg, jpeg, png, gif, webp, svg
  • Documents: pdf, txt, docx, xlsx, pptx
  • Archives: zip, rar, 7z
  • Media: mp4, mp3, wav
Potentially Risky Types:
  • Executables: exe, bat, sh, cmd
  • Scripts: js, php, py, pl
  • System: dll, sys, com
  • Web: html, htm (can contain scripts)
Security Note: Only allow file types you trust. Executable files and scripts can pose security risks.
