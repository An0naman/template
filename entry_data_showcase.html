How It Works
1. Define Metrics

In Settings → Entry Metrics, define named numeric measurements scoped to an entry type.

Temperature (°C) · Brewing Batch
Original Gravity · Brewing Batch
pH · Brewing Batch
2. Record Readings

On any entry's Data Points section, click Add Reading to log a timestamped value for any metric.

Batch #42 → Temperature = 18.4 °C @ Day 1
Batch #42 → Gravity = 1.052 @ Day 1
Batch #42 → Temperature = 17.9 °C @ Day 3
3. Plot & Compare

View charts on the entry itself, or add a Dashboard Widget to compare many entries at once — either via a Saved Search or by following a relationship from a pivot entry.

E.g. "Show temperature of all batches related to Recipe #7"
