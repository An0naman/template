## Fastest Method: ChatGPT (DALL-E)

### Step 1: Go to ChatGPT
Visit: https://chat.openai.com (requires account)

### Step 2: Use These Prompts

**For Nature/Landscape Backgrounds:**
```
Create a high-resolution wallpaper with a serene mountain landscape at sunset, 
soft pastel colors, minimalist style, suitable for an application background, 
subtle details, 4k quality, horizontal orientation 16:9
```

**For Abstract/Tech Backgrounds:**
```

gradient colors from deep blue to purple, modern tech aesthetic, clean, 
suitable for application background, 4k, 16:9 aspect ratio
```

**For Dark Mode Backgrounds:**
```
Create a dark mode wallpaper with subtle nebula effects, deep purples and blues, 
starfield, minimalist, high resolution, suitable for application UI background,
```

**For Light Mode Backgrounds:**
```
Generate a light, airy wallpaper with soft clouds and pastel sky, 
gentle gradients, minimalist, suitable for application background, 
clean aesthetic, 4k quality, horizontal format
```

### Step 3: Download & Upload
1. Download the generated image
2. Go to your app: **Maintenance** ‚Üí **System Theme** ‚Üí **Background Image**
3. Toggle "Enable Background Image" ON
4. Upload your downloaded image
5. Adjust opacity to 30-50% for readability
6. Add slight blur if needed
7. Save settings

## Tips for Best Results

### ‚úÖ DO Include:
- "high-resolution" or "4k quality"
- "wallpaper" or "background"
- "suitable for application background"
- "minimalist" or "subtle"
- Specific colors matching your theme
- "16:9 aspect ratio" or "horizontal"

### ‚ùå AVOID:
- "busy" or "cluttered" compositions
- Too many small details
- Very bright or high-contrast images
- Text or UI elements in the image
- Very dark images (unless for dark mode)

### Recommended Settings After Upload:
- **Opacity**: 30-50% (good balance)
- **Blur**: 3-5px (softens details)
- **Size**: Cover (fills screen)
- **Position**: Center
- **Parallax**: OFF (better performance)

## Troubleshooting

### Image Too Bright?
- Reduce opacity to 20-30%
- Increase blur to 8-10px
- Try generating a darker version

### Image Too Busy?
- Increase blur significantly (10-15px)
- Reduce opacity to 20-30%
- Generate a simpler, more minimalist version

### Colors Don't Match Theme?
- Specify your theme colors in the prompt
- Example: "using blue #0d6efd and purple #8b5cf6"
- Try different variations

### Performance Issues?
- Compress image using https://squoosh.app
- Convert to WebP format
- Reduce resolution to 1920x1080
