# Documentation Structure

This directory contains all project documentation organized by category.

## Directory Structure

### `/features/`
Active feature documentation and user guides.

### `/features-archive/`
Historical implementation notes, completed features, and archived documentation:
- Feature implementation details
- Completed migration notes
- Bug fix documentation
- Development summaries

### `/bug-fixes/`
Documentation of bug fixes and patches.

### `/development/`
Development and testing documentation:
- Testing guides
- Debug notes
- Analysis documents

### `/framework/`
Framework-related documentation:
- Framework usage guides
- Auto-update system
- Deployment guides

### `/guides/`
User and developer guides:
- Security implementation
- Testing procedures

### `/modules/`
Module-specific documentation.

### `/setup/`
Setup and installation guides:
- CasaOS setup
- Environment configuration

### Root Level Files
- `MIGRATIONS.md` - Database migration documentation
- `MIGRATIONS_QUICK_REF.md` - Quick reference for migrations
- `MIGRATION_SYSTEM_IMPLEMENTATION.md` - Migration system details

## Navigation Tips

- For current features and capabilities, check `/features/`
- For setup instructions, check `/setup/`
- For development work, check `/development/` and `/guides/`
- For historical context on implementations, check `/features-archive/`
